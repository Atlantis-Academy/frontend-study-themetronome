Подключенный в `index.html` скрипт на первый взгляд не делает ничего, так как все "действия" разбиты по функциям. Для вызова функции записи общего количества просмотренных фильмов, откройте консоль разработчика и во вкладке `console` и вызовите саму функцию - `initialAmountCheck();`. Данная функция запишет в одно из полей объекта введенное вами значение. Так же доступны функции для записи последних просмотренных фильмов(и их оценки) - `recordFilmsInformation();`. Вывод вашего "уровня киномана" - `consoleViewerLevel();`. Изменение поля приватности объекта `personalMovieDB` - `showMyDB();`. Запись трёх любимых жанров - `writeYourGenres();`.
